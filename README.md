# Invoice Management System

## Description
This is a Python-based Invoice Management System that allows users to:
- Manage customers
- Track products and stock
- Generate invoices
- Manage users

The system uses a MySQL/XAMPP database to store all records.

## Features
- Add, update, delete customers, products, and invoices
- View all tables in one place
- Python loops and database integration for easy data management
- Fully functional CRUD operations

## How to Run
1. Install Python 3.x
2. Install MySQL/XAMPP and create the database
3. Run the Python scripts in PyCharm or any IDE
